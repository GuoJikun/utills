# Utils 工具函数

提供常用的函数方法


## Array 方法

|方法名| 描述| 参数| 返回值|
|---|---|---|---|
|sum| 求和| `(Array<T extends object | number>)` | `number` |
|average| 求平均值| `(Array<number>)` | `number` |
|min| 最小值| `(Array<number>)` | `number` |
|max| 最大值| `(Array<number>)` | `number` |
|isEmpty| 是否为空| `(Array<any>)` | `boolean` |
